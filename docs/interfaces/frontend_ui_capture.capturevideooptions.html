Optional estimatePoses

estimatePoses: undefined | boolean
  • Defined in frontend/ui/capture/index.ts:142

Generates pose detection data. When enabled, pose data is available after upload by appending .poses to the returned file URL. For example, https://objects.koji-cdn.com/project-id/my-video.mp4.poses.

Optional hideExtensions

hideExtensions: undefined | boolean
  • Defined in frontend/ui/capture/index.ts:144

Whether to hide all asset packs and VCC extensions. Enable this option in cases where they do not make sense (for example, Kojis for selling premium videos).

Optional hls

hls: undefined | boolean
  • Defined in frontend/ui/capture/index.ts:140

Enables HTTP Live Streaming (HLS) for delivery of longer content. When enabled, uploaded videos are transcoded for HLS and return an m3u8 playlist. Use this feature in conjunction with hls.js for controlling playback.
