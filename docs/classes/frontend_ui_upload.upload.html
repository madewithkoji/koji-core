uploadFile

  • uploadFile(options: UploadOptions): Promise<null | string>
    • Defined in frontend/ui/upload/index.ts:61

    Uploads a file to your project’s CDN. You can use this method to upload media created or captured by the user as part of the template experience. For example, recording a video or drawing on a canvas.

    example
    const url = await Koji.ui.upload.uploadFile({
 file: new File([blob], 'video.webm'),
 type: 'video',
 videoOptions: {
   hls: true,
 });

    Parameters

    • options: UploadOptions

      Request options for the file upload.

    Returns Promise<null | string>

    Unique URL for accessing the file on images.koji-cdn.com or objects.koji-cdn.com, depending on the type of file.
