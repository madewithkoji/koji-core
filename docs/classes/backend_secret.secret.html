generateSignedUrl

  • generateSignedUrl(resource: string, expireSeconds?: number): Promise<string>
    • Defined in backend/secret/index.ts:87

    Generates a signed URL for securely serving a file, without exposing the permanent URI. This method provides additional security against sharing or rehosting of protected content.

    example
    const temporaryImagePath = await secret.generateSignedUrl('https://images.koji-cdn.com/e83eaff0-279f-4403-951b-e56507af923d/userData/emfga-icon.png');

// Blur the image
const temporaryBlurredImagePath = await secret.generateSignedUrl('https://images.koji-cdn.com/e83eaff0-279f-4403-951b-e56507af923d/userData/emfga-icon.png?blur=10');

    Parameters

    • resource: string

      Full URL of the resource. If the resource is an image hosted on the Koji CDN, you can pass in query parameters to transform it. For example, resize or crop the image.

    • Optional expireSeconds: number

      Amount of time for which the signed URL remains valid, in seconds. If undefined, signed videos expire after 1 hour, and any other resource expires after 5 minutes.

    Returns Promise<string>

    Signed URL for the resource.
